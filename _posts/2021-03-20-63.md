```c
 for (int i = 0; i < 5; i++)
    {
        for (int j = i; j < 5 - 1; j++)
        {
            printf("O");
        }
        for (int k = 0; k <=i; k++)
        {
            printf("*");
        }
        printf("\n");
    }    
```
위 코드의 결과값은 다음과 같다

> OOOO*  
> OOO**  
> OO***  
> O****  
> *****  

그리고 위의 결과를 동일하게 출력하는 코드를 다른 방식으로 몇 가지 적어보았다.
